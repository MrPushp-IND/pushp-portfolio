# Pushp Raj — Data Analytics Portfolio

> **Live site:** https://mrpushp-ind.github.io/pushp-portfolio/

My personal portfolio website — a single-page, dark terminal-style showcase of five data analytics projects built on **real Indian company financials**:

1. **Apollo Hospitals FY25** — 5-page Power BI financial performance report (18+ DAX measures, star schema)
2. **ITC Limited FY25** — SQL segment profitability & capital allocation analysis ([repo](https://github.com/MrPushp-IND/itc-segment-profitability-sql))
3. **Britannia** — Python SARIMA demand forecasting, ~8% MAPE ([repo](https://github.com/MrPushp-IND/britannia-demand-forecasting-python))
4. **Maruti Suzuki FY25** — executive Power BI cockpit, 25+ DAX measures ([repo](https://github.com/MrPushp-IND/maruti-executive-cockpit-powerbi))
5. **HR Analytics** — attrition dashboard on 1,572 employee records

## Tech

Plain HTML/CSS/JS — no frameworks, no build step. Hosted on GitHub Pages.
